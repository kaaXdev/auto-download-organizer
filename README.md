# 🧰 Auto Organizer (Windows)

Simple background tool that automatically organizes your Downloads folder.  


---

## 🚀 What it does

- 📂 Sorts files automatically from Downloads:
  - Images
  - Videos
  - PDFs
  - Archives (ZIP, RAR, 7Z)
  - Installers (EXE, MSI)
- 🔄 Runs automatically on system startup
- 🧠 Waits until files are fully downloaded before moving them
- 🔇 Runs silently in background (no visible window)
